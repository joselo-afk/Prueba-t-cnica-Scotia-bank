Ejecucion del codigo: 

Un comando de ejcucion normal de un proyecto de java desde 
el archivo main y a partir de esto utilizar las diferentes
opciones permitidas para el desarrollo de la aplicacion, 
en su final de su funcionamiento solo presionar la tecla
 0 para salir.

Para el uso de las pruebas unitarias solo es cuestion de 
utilizar el comando "mvn test" mediente la consola de comandos
o mediante algun otro plugin de ejecucion de pruebas mediante 
JUnit.
